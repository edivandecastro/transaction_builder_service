# Dependencies

This Service uses and depends on the following technologies:

### Local installation required

Rails ~> 7.0.8
Ruby     3.2.1
Docker   24.0.6

### Used through docker

MongoDB

# Getting Started

To start the application, run docker with the MongoDB image

```
docker compose up -d
```

```
rails db:seed && rails s
```
